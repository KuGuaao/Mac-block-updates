# Mac-block-updates
一键屏蔽 macOS 系统更新、清除小红点的工具，支持 macOS 12+ 到 15 Sequoia。纯脚本运行，无任何安全提示。
一键使用教程（小白版）
第一步：打开终端
在 Mac 里找到 启动台 → 其他 → 终端，打开它
第二步：复制这行命令，粘贴到终端里


curl -sSL https://raw.githubusercontent.com/KuGuaao/Mac-block-updates/main/update-blocker.sh | sudo bash


粘贴后按 回车
第三步：输入密码
输入你的电脑开机密码（输入时不显示文字，正常输完回车就行）
第四步：选择功能（点一下就行）
会弹出一个图形窗口，4 个功能任选：
一键屏蔽所有系统更新（完全禁止更新）
一键恢复系统更新（想更新了用这个）
清除更新小红点（去掉设置上的红点）
只屏蔽大版本，保留安全更新
点一下你要的功能，再点好，就自动完成了！
