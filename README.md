# Mac-block-updates
一键屏蔽 macOS 系统更新、清除小红点的工具，支持 macOS 12+ 到 15 Sequoia。纯脚本运行，无任何安全提示。
# Mac 屏蔽系统更新
一键屏蔽 macOS 更新 / 清除小红点

## 🖥 一键使用（复制到终端运行）
```bash
curl -sSL https://bit.ly/3WnqXfU | sudo bash
功能
・屏蔽所有系统更新
・恢复系统更新
・清除设置小红点
・仅屏蔽大版本更新
运行后输入电脑密码，点选功能即可使用
